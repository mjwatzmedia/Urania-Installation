Grid Context Visualizations

These graphics translate regional electricity data into an accessible framework for understanding how large-scale computing loads interact with Tucson’s power system (TEPC). They emphasize patterns and scale, not prediction.

About these images ⌄

Using publicly available TEPC grid data, the visuals show how demand, imports, and system stress vary across the year, and how an illustrative data-center-scale load would sit within those existing rhythms.

The import charts highlight seasonal reliance on outside power during high-demand months. The shortfall comparison shows how an added, constant load amplifies seasonal stress. The load duration curve places that added demand against the energy mix and peaker needs to make the infrastructure implications legible for a general audience.

Together, these images support Urania by making visible the physical infrastructure behind “cloud” computation: electricity, fuel, water, land, and civic capacity.
