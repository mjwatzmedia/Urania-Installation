Pepper’s Ghost Stage · 3D TV Shell

The Pepper’s Ghost “hologram” uses a stage with an acrylic panel + a TV shell, so the phone image appears to float in space.

Files you’ll need:

  • 3D models for the TV case (e.g., .stl or .3mf)
  • 3D models for the internal stage / phone holder
  • Laser-cut dimensions for the clear acrylic panel

3D files for the illusion stage are here:
Illusion Stage · 3D models on GitHub

Build Steps (Summary)

  1. Print the TV shell and stage pieces on a 3D printer. Assemble.
  2. Cut the acrylic panel according to the dimensions. Mount the acrylic panel inside the stage.
  3. Test fit your smartphone so that the reflected hologram is centered in the acrylic.
  4. Load the Urania hologram loop on the phone, set brightness high, and place the phone into the stage.

Raspberry Pi 3 · Projector Loop Setup

This setup uses a Raspberry Pi 3 connected to a projector to auto-play the Urania teaser scene in a continuous loop when the Pi boots.

Hardware

  • Raspberry Pi 3 (or similar)
  • microSD card with Raspberry Pi OS
  • HDMI cable + projector or HDMI display
  • Power supply for the Pi

Basic Setup Steps

  1. Install Raspberry Pi OS on the microSD card and boot the Pi.
  2. Copy the Urania teaser video file onto the Pi (e.g., to /home/pi/videos/urania-teaser.mp4).
  3. Install a lightweight media player (for example, omxplayer or vlc).
  4. Create a small startup script that:
    • waits a few seconds after boot, then
    • launches the media player in full-screen loop mode on the projector.
  5. Add that script to your autostart (for example via ~/.config/lxsession or crontab with @reboot).

Once this is configured, you should be able to plug in the Pi and projector, and the teaser will start playing automatically in a loop after boot.
