Overview

This site hosts everything needed to build and exhibit the Urania installation:

  • 3D-printed, smartphone-compatible Pepper’s Ghost stage + TV shell
  • Raspberry Pi 3 projector loop that auto-plays a teaser scene on boot
  • Two teaser videos:
    • Mirrored hologram loop for the Pepper’s Ghost stage
    • Teaser scene from the Urania pilot for the wall projector

Basic workflow:
QR code → opens this site → you choose the hologram or the teaser →
Phone goes into the Pepper’s Ghost stage · Raspberry Pi 3 drives the projector.

For detailed build steps for the stage and Raspberry Pi setup, see the Urania build & setup guide.

About the Project

Urania is a dreamcore-inspired AI installation and part of an ongoing investigation into human–machine relationships, genre aesthetics, and the environmental cost of technological evolution. This page is a practical hub for anyone exhibiting or rebuilding the installation.
